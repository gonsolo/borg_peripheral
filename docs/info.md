# Floating point adder

Author: Andreas Wendleder

Peripheral index: nn

## What it does

It adds two floating point values.

## Register map

No registers are used.

| Address | Name  | Access | Description                                                         |
|---------|-------|--------|---------------------------------------------------------------------|
| 0x00    | DATA  | R/W    | A word of data                                                      |

## How to test

There is a cocotb test that adds two numbers.
